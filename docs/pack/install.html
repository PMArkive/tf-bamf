Installing BAMF

There are multiple ways to install BAMF:

  • Use the installer - Coming soon!
  • Basic Manual installation (see below)
  • Advanced Manual installation

It is recommended that you read about the Features first. If you don't want a module right now, you can always install it later.

Basic Manual Installation

This guide is intended for absolute beginners. It should contain anything you need to get you mapping as quickly as possible, but leaves out advanced modules like compile pre-sets or tools.

  1. Download the latest version of BAMF from TF2Maps.net. The content of the .zip should look like this:
  2. Navigate to your Team Fortress 2 installation and open it in another window. (In Steam, right-click on Team Fortress 2 and select Manage -> Browse Local Files. You're going to want to bookmark this folder as it is rather essential for mapping.)
  3. In BAMF, open the FGD folder. Move the entire content into Team Fortress/bin.
    Note: There is another bin folder in Team Fortress 2/tf. Ignore that one, it is never used for anything mapping-related.
  4. In BAMF, open the Prefabs folder. Move the entire content into Team Fortress/bin/Prefabs. If that directory doesn't exist, create it.
  5. In BAMF, open the VPKs folder. Move the entire content into Team Fortress/tf/custom.
    This is where all your custom mapping assets and other TF2 mods are stored.
  6. Open Hammer (or Hammer++). Navigate to Tools → Options..., then select the Game Configurations tab.
    Delete the existing entry in the Game Data files section by selecting and clicking Remove.

    Then add the new Game Data File by clicking Add and navigating to Team Fortress 2/bin/tf-brokk.fgd.

    7. That's it! You can now start mapping.
