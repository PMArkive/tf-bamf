Installing BAMF

There are multiple ways to install BAMF:

  • Use the Installer
  • Basic Manual installation
  • Advanced Manual installation

It is recommended that you read about the Features first. If you don't want a module right now, you can always install it later, either manually or by running the installer again.

Using the Installer

While the automated installer prevents you from having to manually move files around like a bloody caveman, there are still a few things you need to know.
Special thanks to Feesh, Mystyc Cheez, SED, and Nick from the TF2Maps Discord for alpha testing.

  1. If you haven't already, grab the latest version of the BAMF Installer from TF2Maps.net or from GitHub.
  2. Run the installer and accept the license. The next page will give you a quick overview of the pack.
  3. The next three pages will ask you for several file paths.
    • TF2 installation:
      Should be pretty clear.
    • Map files:
      Where you want to place reference VMFs from the Game Mode Module and the VMF Modules.
      If you do not want to install these, leave the path in the default location. Nothing will happen to it.
    • CompilePal installation:
      The location of CompilePal, if you have it installed. This should be the folder that contains CompilePalX.exe and various other folders.
      If you don't have CompilePal or don't want the CompilePal Sequences Modules or the error parser, leave the path in the default location. Nothing will happen to it.
  4. Configure which modules you would like to install. Note that some modules have requirements, as stated on the Features page.
  5. Confirm your selection and finish the installation.
  6. If you have installed the FGD Module:
    Open Hammer (or Hammer++). Navigate to Tools → Options..., then select the Game Configurations tab.
    Delete the existing entry in the Game Data files section by selecting and clicking Remove.

    Then add the new Game Data File by clicking Add and navigating to Team Fortress 2/bin/tf-brokk.fgd.
